# Le répertoire chaud
Ce site n'est **pas** un factsheet qui a vocation à être partagé, c'est juste une compilation *personelle* de documents potentiellement intéressants dans des sujets divers, de qualité variable (certains documents de mauvaise qualité peuvent même servir d'exemple pour prouver quelque chose). Néanmoins, toute contribution est encouragée.
